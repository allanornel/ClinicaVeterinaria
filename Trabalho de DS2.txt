Ordem do Trabalho:
Desenvolvimento de um Sistema JAVA Desktop que apresente as seguintes características e/ou operações:
- uma interface e/ou uma classe abstrata
- uma herança.
- uma composicao/agregação
- uma associação simples.
- um método que foi alvo de polimorfismo
- CRUD das entidades/classes mais importantes (ou de todas).
- A lógica proposta pelo sistema (Ex: aluguel de dvds e etc)
- Divisão entre apresentação/negócio/persistencia
- Pelo menos uma instrução SQL com inner join (adicionar no DAO)
- Pelo menos uma instrução SQL com group by (adicionar DAO)
- Pelo menos uma instrução SQL com funções de agregação: avg, sum e etc. (adicionar no DAO)
- Utilizar e pesquisar sobre Padrões de Projeto (singleton (conexão bd, factory (para os daos) e etc)
==================================================
Observações:
- A interface pode ser pelo Terminal
- O sistema pode ser uma loja, uma locadora, uma garagem coletiva, um cadastro de super-heróis ou etc...
